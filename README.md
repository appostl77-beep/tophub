# TopHub - DLE Template для сайта рейтингов

Современный адаптивный шаблон для DataLife Engine, специально разработанный для сайтов пользовательских рейтингов и обзоров.

## 🎨 Особенности дизайна

- **Светлая пастельная цветовая схема** - серый, голубой-синий
- **Акцент на контенте** - минималистичный дизайн
- **Полная адаптивность** - работает на всех устройствах
- **Кроссбраузерность** - поддержка всех современных браузеров

## 🚀 Установка

1. Скопируйте папку `tophub` в директорию `templates/`
2. В админ-панели DLE выберите шаблон "TopHub"
3. Настройте кастомные поля для рейтингов:
   - `rating_gallery` - галерея изображений
   - `rating_specs` - характеристики
   - `rating_links` - полезные ссылки

## ⚙️ Настройка

### Цветовая схема
Измените CSS переменные в `css/styles.css`:

```css
:root {
  --primary: hsl(213, 90%, 35%);      /* Основной цвет */
  --background: hsl(210, 40%, 98%);   /* Фон */
  --foreground: hsl(222, 39%, 12%);   /* Текст */
  /* ... другие переменные */
}
```

### Логотип
Замените файл `images/logo.png` на свой логотип.

### Шрифты
По умолчанию используется шрифт Inter. Для изменения отредактируйте импорт в `css/styles.css`.

## 📱 Адаптивность

Шаблон полностью адаптивен и оптимизирован для:
- Десктопов (1200px+)
- Планшетов (768px - 1199px)
- Мобильных устройств (до 767px)

## 🔧 Кастомизация

### Добавление новых страниц
1. Создайте новый `.tpl` файл
2. Используйте структуру из существующих файлов
3. Подключите в `main.tpl` через `{include}`

### Модификация стилей
- Основные стили: `css/styles.css`
- Специфичные стили: встроены в `.tpl` файлы
- Переменные: в начале `css/styles.css`
